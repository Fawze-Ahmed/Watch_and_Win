# ALF Web MVP

Free, publish-ready web version of the project with:

- internal points instead of direct cash or crypto payouts
- earning tasks
- reward requests
- live support chat
- owner dashboard
- store placeholder marked as coming soon

## Run locally

```bash
npm install
npm run dev
```

## Current stack

- Vite
- TypeScript
- Supabase
- Vercel-ready frontend build

## Current product scope

- users earn internal points
- users submit internal reward requests
- users can send support messages
- owner can review and reply from the dashboard

## Important note

This public version is intentionally positioned as an internal points platform for a safer free launch.
Do not treat it as a production cashout or crypto payout system yet.

## Deployment

1. Push the project to GitHub.
2. Import the repo into Vercel.
3. Add the environment variables from `.env.local` into Vercel Project Settings.
4. Deploy.
